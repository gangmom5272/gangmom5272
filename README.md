## 🙋‍♂️ About Me

안녕하세요! 제 GitHub 프로필에 방문해 주셔서 감사합니다 😊

저는 **한국외국어대학교 프랑스어과**를 졸업하고, 현재 **백엔드 개발자**로의 전환을 목표로 열심히 성장 중입니다.
SSAFY 15기 과정을 통해 Python, Django, FastAPI 등을 배우며 꾸준히 실력을 쌓고 있어요.

언어를 공부하면서 길렀던 논리적 사고와 커뮤니케이션 능력을 바탕으로,
**데이터와 로직으로 문제를 해결하는 백엔드 개발자**가 되고 싶습니다.

코드로 세상을 연결하는 개발자가 되는 그날까지, 오늘도 한 줄씩 쌓아가고 있습니다 🚀
